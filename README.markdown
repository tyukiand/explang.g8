A [Giter8][g8] template for small programming language experiments

Automatically sets up some basics, such as

  * `cats`
  * `cats-mtl`
  * `cats-effect`
  * `scala-parser-combinators`
  * `antlr4` already set-up (remove one if you don't need both)
  * `scalacheck`
